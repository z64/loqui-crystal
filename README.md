# loqui

Crystal port of the [loqui RPC protocol](https://github.com/discordapp/loqui)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     loqui:
       github: your-github-user/loqui
   ```

2. Run `shards install`

